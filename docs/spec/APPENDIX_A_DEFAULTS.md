# 부록 A. 기본 테이블 및 기본값 규칙 (Defaults & Tables)

FHDL MVP(v0.1)는 사용자가 모든 설계 파라미터를 입력하지 않아도 합리적인 설계가 가능하도록 본 부록에 정의된 표준 규칙과 기본값을 사용합니다.

## 1. 기본 물성 및 단위 규칙

### 1.1 유체 (Water 기준)
*   **밀도($\rho$):** 1,000 kg/m³
*   **중력가속도($g$):** 9.81 m/s²
*   **특이사항:** 초기 버전은 물 전용이며, 다른 유체 지정 시 경고를 출력합니다.

### 1.2 단위 정규화 (Normalization)
사용자가 입력한 모든 단위는 계산 전 아래의 SI 표준 단위로 즉시 변환됩니다.
*   **길이/수두:** m (입력: mm, cm, m, ft 지원)
*   **유량:** m³/s (입력: LPM, L/s, GPM 지원)
*   **압력:** Pa (입력: MPa, bar, psi 지원)

## 2. 설계 제약 기본값 (Default Constraints)

사용자가 `constraint` 블록을 정의하지 않았을 때 적용되는 기본값입니다.

| 항목 | 기본값 | 설명 |
| :--- | :--- | :--- |
| **최소 허용 유속 ($V_{min}$)** | 0.6 m/s | 침전 방지 및 정체 구간 방지용 |
| **최대 허용 유속 ($V_{max}$)** | 2.5 m/s | 소음 및 수충격 방지용 |
| **양정 여유율 ($SF_H$)** | 1.1 (10%) | 계산 오차 및 노후화 대비 |
| **탱크 안전율 ($SF_T$)** | 1.1 (10%) | 증발 및 비상 용량 대비 |
| **탱크 사용률 ($u$)** | 1.0 (100%) | MVP 단순화 (추후 0.9 적용 권장) |

## 4. 마찰 및 국부 손실 규칙

### 4.1 재질별 기본 마찰계수 ($f$)
사용자가 마찰계수를 지정하지 않은 경우 적용되는 Fallback 값입니다.
*   **PVC:** 0.020
*   **Steel (ASTM A53):** 0.030
*   **Default Fallback:** 0.025

### 4.2 국부 손실 (Minor Loss)
*   사용자가 `loss_k`를 명시하지 않은 경우 기본값 **0.0**을 적용합니다.
*   단, 리포트에는 "국부 손실 미반영"을 명시하여 설계자가 인지하게 합니다.

## 5. 적용 우선순위 (Precedence)

1.  **사용자 직접 지정값** (개별 구성요소 블록 내)
2.  **프로젝트 전역 제약** (전역 `constraint` 블록 내)
3.  **부록 A 기본 테이블 및 Fallback 값**
4.  **계산 불가 판정 및 에러 고지**
