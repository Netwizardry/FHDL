# 03. 결함 등급 판정 기준 (Severity Criteria)

| 등급 | 코드 | 설명 | 판정 예시 |
| :--- | :--- | :--- | :--- |
| **Critical** | **CRT** | 구현 중단 및 시스템 붕괴 | 물리 공식 충돌(DW vs HW), 데이터 유실 가능성 |
| **Major** | **MAJ** | 출시 불가 및 주요 기능 결함 | 인터페이스 파라미터 미비, 필수 보안 요구사항 누락 |
| **Minor** | **MIN** | 품질 저하 및 운영 불편 | 모호한 표현("적절히"), 예외 케이스 설명 부족 |
| **Trivial** | **TRI** | 단순 관리 소홀 | 문서 번호 오기, 오타, 서식 불일치 |
