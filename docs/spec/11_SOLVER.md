# 12. 솔버 엔진 (Solver Engine) 명세

FHDL 솔버 엔진은 관망의 물리적 타당성을 검토하고, 유량과 압력의 평형 상태를 계산하기 위해 2-Pass 수치 해석 알고리즘을 사용합니다.

## 1. 2-Pass 해석 알고리즘

### 1.1 Pass 1: 설계 역산 (Synthesis)
*   **목적:** 말단 장치(Terminal)의 요구 사양으로부터 상류로 거슬러 올라가며 필요한 유량과 최소 수두를 추정합니다.
*   **프로세스:**
    1.  말단 노드(Terminal)에서 시작하여 상류(Source/Tank) 방향으로 그래프 탐색.
    2.  노드 유량 합산: $Q_{upstream} = \sum Q_{downstream} + Q_{node\_demand}$.
    3.  단순 손실 추정: 정적 손실(고도차)과 마찰 손실의 기초값을 합산하여 상류 노드의 필요 수두 계산.
*   **특징:** 복잡한 반복 계산 전, 시스템의 대략적인 규모와 연속성을 확인하는 Pre-flight 체크 역할을 합니다.

### 1.2 Pass 2: 동적 검증 (Verification)
*   **목적:** Newton-Raphson 반복법을 사용하여 전 관망의 에너지 평형과 질량 보존을 만족하는 최종 상태를 산출합니다.
*   **핵심 원리:**
    *   **질량 보존 (Continuity):** 각 노드에서 $\sum Q_{in} - \sum Q_{out} = 0$.
    *   **에너지 평형 (Energy):** 폐회로 또는 두 점 사이의 수두차 $\Delta H = h_f + h_m$.
*   **수치 해석 기법:**
    *   **Damping Factor:** 급격한 변동을 방지하기 위해 보정값에 0.3~0.5의 감쇠 계수 적용.
    *   **Tolerance:** 최대 수두 변화량이 $10^{-6}$ m 이하일 때 수렴한 것으로 간주.

## 1.3 물성 전파 엔진 (Context Propagation - M01)
시스템은 복잡한 배관망에서 상류의 물성(온도, 유체 종류)이 하류로 일관되게 전달되도록 합니다.
*   **Propagation Rule:** 각 노드는 인입(Inlet) 유동의 물성 정보를 합산(Weighted Average)하여 자신의 물성으로 확정하고, 이를 하류 엣지로 전파합니다.
*   **Static Property Assumption:** v0.1에서는 전체 시스템 온도가 균일하다고 가정하나, 내부 데이터 구조는 노드별 개별 온도($T_{node}$)를 보유하여 향후 확장이 가능하도록 설계합니다.

### 2.1 배관경 선정 (Pipe Sizing)
사용자가 관경을 `auto`로 지정한 경우, 다음 절차를 따릅니다.
1.  **구간 유량 집계:** 말단 장치의 요구 유량을 상류로 합산.
2.  **허용 유속 범위 적용:** 설계 제약(예: 1.0 ~ 2.5 m/s) 내의 유속을 만족하는 관경 탐색.
3.  **표준 관경 테이블 매칭:** 라이브러리(`LibraryManager`)의 표준 규격 중 조건을 만족하는 최소 관경 선정.
4.  **최종 결정:** 유속, 마찰 손실, 비용 효율성을 고려하여 권장 관경 확정.

### 2.2 펌프 사양 산정 (Pump Sizing)
시스템은 펌프의 최소 요구 성능을 다음 합산값으로 정의합니다.
*   **요구 유량 ($Q_{req}$):** 모든 말단 장치 유량의 합 × 여유율.
*   **요구 양정 ($H_{req}$):** 정수두(고저차) + 배관 마찰손실 + 국부손실 + 말단 요구 압력 환산 수두.

### 2.3 탱크 용량 산정 (Tank Sizing)
저수조 크기는 시스템 가동의 연속성을 보장하기 위해 다음 기준으로 산정합니다.
*   **공식:** $V_{tank} = \sum Q_{total} \times T_{target} \times Margin$
*   $T_{target}$: 목표 연속 운전 시간 (기본값: 30분).

## 3. 주요 물리 공식

### 2.1 마찰 손실 (Friction Loss)
*   **Darcy-Weisbach 공식:** $h_f = f \cdot \frac{L}{D} \cdot \frac{v^2}{2g}$
*   **마찰계수(f):** 레이놀즈 수($Re$)에 따라 구분.
    *   층류 ($Re < 2000$): $f = 64 / Re$
    *   난류 ($Re > 4000$): Churchill 또는 Colebrook-White 근사식 사용.
    *   천이 구역 ($2000 \leq Re \leq 4000$): 3차 스플라인 보간으로 불연속성 제거.

### 2.2 국부 손실 (Minor Loss)
*   **공식:** $h_m = \sum K \cdot \frac{v^2}{2g}$
*   **K-계수:** 밸브, 엘보우, 분기점 등의 형상에 따른 저항 계수.

### 2.2 펌프 및 말단 장치 방정식 (C01)
시스템의 비선형 평형 상태를 계산하기 위해 다음 방정식을 사용합니다.
*   **펌프 양정:** $H_{out} = H_{in} + f(Q_{actual})$ (Q-H 곡선)
*   **말단 오리피스:** $Q_{terminal} = K \cdot \sqrt{P_{terminal} / \rho g}$
    *   $K$: `k_factor` (방전 계수 포함). 압력이 0 이하일 경우 유량은 0으로 고정.

### 2.3 대기압 경계 조건 (Atmospheric Boundary - C02)
*   **적용:** 월류보(Weir), 수조 수면, 자유 배출구.
*   **로직:** 해당 노드의 $P_{gauge} = 0$ 조건을 시스템 행렬의 경계값으로 강제 주입하여 개수로-압력관 경계 해석.

## 4. 동적 해석 및 수충격 로직 (Time-dependent Analysis)

시스템은 `Step` (시간 단계) 변수를 사용하여 급격한 밸브 폐쇄 시의 과도 응답을 시뮬레이션합니다.

### 4.1 수충격 압력파 전파 (Water Hammer)
*   **압력파 속도 ($a$):** $a = \sqrt{\frac{K/\rho}{1 + (K/E)(D/e)}}$
    *   $K$: 유체의 체적탄성계수, $E$: 배관의 탄성계수, $e$: 관벽 두께.
*   **Joukowsky 압력 상승:** $\Delta P = \rho \cdot a \cdot \Delta V$
    *   $\Delta V$: `Step` 시간 동안의 유속 변화량.

### 4.2 수치 적분 (Numerical Integration)
밸브 폐쇄 시간($T_c$) 동안 각 시간 단계($\Delta t = Step$)마다 다음을 계산합니다.
1.  **유속 감소량 계산:** $V_{t+\Delta t} = V_t \cdot (1 - \Delta t/T_c)$
2.  **압력 상승 합산:** 각 구간의 압력파 중첩(Superposition)을 고려하여 절점별 최대 압력($P_{max}$) 산출.
3.  **안전 검토:** $P_{max}$가 배관 및 기기의 설계 압력을 초과할 경우 `WRN004` 발생.
